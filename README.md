# kemasinaja - Static Website (Edible Film)
Website statis siap dipublish ke GitHub Pages.

## Cara pakai
1. Edit `script.js` dan ganti WA_NUMBER dengan nomor WhatsAppmu (format internasional, contoh: 6281234567890).
2. Ubah `edible-film.jpg` dengan gambar produk asli.
3. Upload semua file ke repository GitHub (root).
4. Aktifkan GitHub Pages (Settings → Pages → Branch: main / root).
5. Buka `admin.html` di browser (lokal atau hosting) untuk melihat pesanan yang tersimpan di browser seller (localStorage).

## File
- index.html — halaman utama & popup checkout
- style.css — styling
- script.js — logika checkout + kirim WA + simpan order ke localStorage
- admin.html, admin.js, admin.css — dashboard lokal untuk melihat pesanan
